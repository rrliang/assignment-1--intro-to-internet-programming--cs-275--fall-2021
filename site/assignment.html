Assignment

This assignment was to make a website based on Dieter Ram's Ten Principles of Good Design

The two different fonts used on this website were: Itim (cursive) and Walter Turncoat (cursive).

The navigation system included on this website are the different pages of Ten design rules, a quick blurb on Dieter Ram, and this page which holds the specifics of the assignment.

The assignment required to use at most two different colors, and neutral colors. This website used:

Neutrals (shades of blue and white):

  • Body background color: rgb(54, 79, 107);
  • Body text color: rgb(240, 255, 243);
  • Header: rgb(38, 56, 77);
  • Image border: rgb(225, 252, 246);
  • Color of selected Nav page: rgb(13, 152, 161);
  • Text of nav page: rgb(230, 255, 249);
  • Text of accordian element: rgb(255, 255, 255);
  • Color of accordian when hovering and active: rgb(12, 133, 139);

Two different colors (pop of blue and red):

  • Nav and Accordian inactive color: rgb(17, 203, 215);
  • Footer: rgb(250, 70, 89);

Elements and resources used on this website:

  • Anchors and linking, learned during class. Changed color of the link to better fit theme.
  • Accordion element using button script (inspired by the syllabus). Modified the code with own colors, padding, margins, and icons.
  • CSS Box Model, learned in class. Also solidified knowledge on box model here.
  • Content property, used for putting icons (hand-drawn) on the accordion elements.
  • Transforming properties to rotate my icon after the accordian was clicked.
  • Nav tag for quick confirmation of syntax.
  • Formatting elements inline so the nav pages would be on the same line.
  • Wrapping text around an image so the Dieter Rams page would format nicely.
  • Indenting to make paragraphs distinguishable in Dieter Rams page.
  • Border styles in CSS for fancier border (used inset border style at one point).
  • Adding an icon for lists so this list would look nice and fit theme.
  • W3 Validator to check if website is in W3 compliance for both CSS and HTML.
  • Classes clarification to fix previous W3 violations.
  • Align was deprecated so it violated W3 code, changed to Float: in CSS.
